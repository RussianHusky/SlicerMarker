Программа для генерации штриховки внутри прямоугольного контура.

Ввод передаётся через stdin. Пример: hatch_generator.exe --angle 45 --step 1 < rect.txt
Вводной следует структуре:
4
0 0
10 0
10 10
0 10

Если нужна SVG-визуализация, добавляется аргумент: --svg "название файла"
Пример: hatch_generator.exe --angle 45 --step 1 --svg output.svg < rect.txt
